## 🔧 技术实现

### 地图更新机制
```tsx
// 使用时间戳强制刷新
const timestamp = Date.now();
setMapLocation({ lat, lng, address, timestamp });

// 在组件中使用 key 强制重新渲染
<iframe key={`map-${timestamp}`} src={osmUrl} />
```

### 布局结构
```css
display: grid;
grid-template-columns: 600px 1fr;  /* 左侧固定，右侧自适应 */
gap: 24px;
```

### 渐变色设计
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```
